Quick Start

1 Clone Repository

git clone https://github.com/turtacn/OpenEAAP.git cd OpenEAAP

Clone the OpenEAAP repository to your local environment

2 Start Dependencies

docker-compose up -d

Start PostgreSQL, Redis, Milvus, MinIO via Docker Compose

3 Migrate Database

make migrate-up

Initialize database schema with migration scripts

4 Configure Env

cp configs/development.yaml.example configs/development.yaml

Copy and modify configuration file for your environment

5 Start Service

make run-server

Start HTTP service (http://localhost:8080)

6 Run Tests

make test

Run all tests to verify the environment
